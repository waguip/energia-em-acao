# Energia em Ação
[![Static Badge](https://img.shields.io/badge/Jogar-darkgreen)](https://cacatua.itch.io/energia-em-acao)

## Resumo
Um jogo com atividades interativas que ensinam sobre energia limpa e economia de energia em casa. <br/>
Desenvolvido na matéria "Padrões De Projeto Com Extensão" lecionada por Simone Nasser Matos na UTFPR. <br/> 
Aplicado para os alunos da [Associação Artesanal do Excepcional de Ponta Grossa](https://www.assarte.org.br).
